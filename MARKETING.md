# ACF CSS Manager - 마케팅 자료

> **ACF CSS: Advanced Custom Fonts & Colors & Styles Setting Manager**  
> WordPress 사이트의 모든 스타일을 한 곳에서 관리하는 올인원 솔루션

**최종 업데이트**: 2025년 12월 18일  
**버전**: 6.2.0

## 🎯 제품 포지셔닝

### 핵심 가치 제안 (Value Proposition)

> **"코딩 없이 전문가 수준의 웹사이트 디자인"**

ACF CSS Manager는 WordPress 관리자가 CSS 지식 없이도 웹사이트의 색상, 폰트, 버튼, 폼 스타일을 직관적으로 관리할 수 있게 해주는 올인원 스타일 관리 플러그인입니다.

### 타겟 고객

| 세그먼트 | 페인 포인트 | 솔루션 |
|---------|------------|--------|
| **웹 에이전시** | 수십 개 고객 사이트 일일이 관리하기 힘듦 | Partner Hub로 중앙 관제 |
| **프리랜서** | 매번 같은 CSS 작업 반복 | 템플릿 저장/재사용으로 시간 절약 |
| **소규모 사업자** | 개발자 없이 사이트 꾸미기 어려움 | AI가 스타일 자동 생성 |
| **기업 마케터** | 브랜드 가이드라인 일관성 유지 어려움 | 팔레트/타이포 잠금 기능 |

## 🎨 비주얼 가이드라인

### 브랜드 컬러

| 용도 | HEX | 설명 |
|------|-----|------|
| Primary | `#2563eb` | 신뢰감 있는 블루 |
| Secondary | `#f59e0b` | 활력 있는 앰버 |
| Success | `#10b981` | 긍정적 그린 |
| Dark | `#1e293b` | 텍스트/배경 |

### 로고 사용

- 전체 로고: "ACF CSS Manager" + 아이콘
- 축약 로고: "ACF CSS" + 아이콘
- 아이콘 단독: 32x32px 이상에서만 사용

### 스크린샷 가이드

1. **Admin Center 메인**: 탭 네비게이션이 보이도록
2. **AI 생성 화면**: 프롬프트 입력 → 결과 Diff 표시
3. **Customizer 연동**: 실시간 미리보기 강조
4. **Partner Hub**: 다중 사이트 관제 대시보드
