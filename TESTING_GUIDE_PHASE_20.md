## 🔍 테스트 방법

### 로컬 WordPress 접속
1. WordPress 관리자 페이지 접속
2. 플러그인 > 설치된 플러그인 메뉴
3. 각 플러그인 활성화

### 언어 변경 테스트
1. 설정 > 일반 > 사이트 언어 변경
2. 각 언어로 변경하여 번역 확인
3. 플러그인 목록 및 관리자 페이지 확인

### 보안 기능 테스트
1. 파일 무결성 모니터:
   - `includes/class-jj-license-manager.php` 파일 일부 수정
   - WordPress 관리자에서 알림 확인
   - 플러그인 기능 제한 확인

2. 라이센스 암호화:
   - 라이센스 키 입력
   - 데이터베이스에서 암호화된 값 확인
   - 복호화 정상 작동 확인
