## 시작하기

### 플러그인 설치 및 활성화

1. WordPress 관리자 페이지로 이동
2. 플러그인 → 새로 추가
3. "ACF CSS" 검색
4. 설치 및 활성화

### 초기 설정

1. 설정 → ACF CSS 관리자 센터로 이동
2. 기본 설정 확인
3. 첫 번째 색상 팔레트 생성

## 기본 기능

### 색상 팔레트 관리

**팔레트 생성**:
1. 스타일 센터 → 색상 탭
2. "새 팔레트 추가" 클릭
3. 색상 추가 및 이름 지정
4. 저장

**팔레트 적용**:
- 팔레트 선택 후 "적용" 버튼 클릭
- 실시간 미리보기로 확인

### 타이포그래피 설정

1. 스타일 센터 → 타이포그래피 탭
2. 폰트 패밀리, 크기, 행간 설정
3. 실시간 미리보기 확인

### 버튼 스타일

1. 스타일 센터 → 버튼 탭
2. Primary, Secondary, Text 버튼 스타일 설정
3. 배경색, 텍스트 색상, 테두리 반경 설정

## 고급 기능

### 버전 관리

**스냅샷 생성**:
1. 관리자 센터 → Tools 탭
2. "스냅샷 생성" 클릭
3. 이름 및 설명 입력
4. 생성

**스냅샷 복원**:
1. 스냅샷 목록에서 복원할 스냅샷 선택
2. "복원" 버튼 클릭
3. 확인

### 스타일 가이드 자동 생성

1. 관리자 센터 → Tools 탭
2. "스타일 가이드 생성" 클릭
3. 포함할 항목 선택
4. 출력 형식 선택 (HTML/Markdown/JSON)
5. 생성

### 전역 검색

- 키보드 단축키: `Ctrl/Cmd + K`
- 모든 설정, 히스토리, 빠른 액션 검색

## AI 기능

### AI 제안 시스템

**실시간 제안**:
- 입력 필드에 타이핑하면 자동으로 제안 표시
- 컨텍스트 기반 스마트 제안

**제안 적용**:
1. AI 제안 패널에서 원하는 제안 선택
2. "제안 적용" 버튼 클릭
3. 자동으로 설정에 반영

**피드백 제공**:
- 좋아요/싫어요 버튼으로 제안 품질 개선

### 자동 최적화

**성능 최적화**:
1. 관리자 센터 → System Status 탭
2. "성능 최적화" 클릭
3. 제안된 최적화 항목 확인
4. 자동 적용 또는 수동 적용

**접근성 개선**:
1. "접근성 개선" 클릭
2. 자동으로 색상 대비, 폰트 크기 조정
3. 포커스 표시 자동 추가

## 문제 해결

### 일반적인 문제

**스타일이 적용되지 않음**:
1. CSS 캐시 확인
2. 브라우저 캐시 삭제
3. 플러그인 재활성화

**성능 문제**:
1. System Status 탭에서 성능 메트릭 확인
2. 성능 최적화 제안 확인
3. 불필요한 스타일 제거

### 에러 해결

**에러 로그 확인**:
1. 관리자 센터 → System Status 탭
2. "에러 로그" 섹션 확인
3. AI 디버거로 자동 해결 방법 확인

**스냅샷으로 복원**:
1. 문제 발생 전 스냅샷으로 복원
2. 자동 스냅샷 확인 (5분 간격)

## FAQ

**Q: 여러 사이트에 동일한 스타일을 적용할 수 있나요?**  
A: 네, 스타일 가이드를 Export/Import 기능으로 공유할 수 있습니다.

**Q: AI Extension이 필요한가요?**  
A: AI 기능을 사용하려면 필요하지만, 기본 기능은 AI Extension 없이도 사용 가능합니다.

**Q: 성능에 영향을 주나요?**  
A: 최적화된 코드와 캐싱 시스템으로 최소한의 영향만 있습니다.
