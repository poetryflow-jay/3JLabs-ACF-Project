# 프레스 릴리즈

## ACF CSS Manager v6.2.0 출시 - WordPress 최초 AI 기반 스타일 관리 플러그인

**즉시 배포 가능**

---

### 요약

J&J Labs가 WordPress 스타일 관리 플러그인 ACF CSS Manager의 최신 버전 6.2.0을 출시했습니다. 이번 버전에는 업계 최초로 AI 기반 스타일 자동 생성 기능이 탑재되어, 사용자가 자연어로 원하는 디자인을 설명하면 AI가 완벽한 색상 팔레트와 타이포그래피를 자동으로 생성합니다.

---

### 본문

**[서울, 2025년 12월 18일]** - WordPress 플러그인 개발사 J&J Labs(대표: Jay & Jenny)가 올인원 스타일 관리 플러그인 'ACF CSS Manager'의 메이저 업데이트 버전 6.2.0을 정식 출시했습니다.

#### AI가 스타일을 디자인하다

이번 버전의 핵심은 **AI 스타일 생성 기능**입니다. 사용자가 "고급스러운 블랙&골드, 법률사무소 느낌"과 같이 원하는 분위기를 자연어로 입력하면, OpenAI의 GPT 기술을 활용해 전문 디자이너 수준의 색상 팔레트, 폰트 조합, 버튼 스타일을 자동으로 생성합니다.

> "CSS를 몰라도, 디자인 감각이 없어도 괜찮습니다. AI에게 원하는 느낌만 말하면 됩니다."
> — Jay, J&J Labs CTO

#### 스마트 프리뷰로 실수 방지

AI가 제안한 스타일은 적용 전에 **Before/After 시각적 비교**를 통해 확인할 수 있습니다. 변경되는 항목만 하이라이트되어 표시되므로, 실수로 잘못된 스타일을 적용할 위험이 없습니다.

#### 클라우드 연동으로 즉시 공유

마음에 드는 AI 생성 스타일은 **원클릭으로 클라우드에 저장**하고, 발급된 공유 코드를 통해 다른 WordPress 사이트에 즉시 복제할 수 있습니다. 향후 템플릿 마켓에서 판매도 가능해질 예정입니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| AI 스타일 생성 | 자연어 프롬프트로 전문가 수준 스타일 자동 생성 |
| 스마트 프리뷰 | Before/After 시각적 Diff 비교 |
| 클라우드 동기화 | 설정 저장 및 다른 사이트로 복제 |
| CSS Tree Shaking | 불필요한 CSS 제거로 성능 향상 |
| Partner Hub | 에이전시를 위한 중앙 관제 대시보드 |
| 멀티사이트 지원 | WordPress 네트워크 전체 스타일 통합 관리 |
