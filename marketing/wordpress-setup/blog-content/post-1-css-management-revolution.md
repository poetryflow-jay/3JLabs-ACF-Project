## 문제: 흩어진 CSS의 악몽

WordPress 사이트를 운영하다 보면 CSS가 여기저기 흩어져 있는 상황을 경험하게 됩니다.

- 테마 커스터마이저의 색상 설정
- 페이지 빌더의 개별 스타일
- 커스텀 CSS 플러그인의 코드
- 자식 테마의 style.css

**결과?** 일관성 없는 디자인, 유지보수의 어려움, 그리고 "이 색상은 대체 어디서 설정한 거지?"라는 끝없는 질문.

---

## 해결책: 중앙 집중식 스타일 관리

ACF CSS Manager는 이 문제를 근본적으로 해결합니다.

### 💡 핵심 개념

```
기존 방식:
테마 설정 + 플러그인 A + 플러그인 B + 커스텀 CSS = 혼란

ACF CSS Manager:
하나의 대시보드 → 전체 사이트 스타일
```

### ✨ 주요 기능

#### 1. 색상 팔레트 관리

```
Primary Color → 모든 버튼, 링크, 강조 요소
Secondary Color → 서브 액션, 보조 요소
Accent Color → 특별 강조, CTA
```

색상을 한 번 변경하면 사이트 전체에 자동 적용됩니다.

#### 2. 타이포그래피 시스템

- Heading 폰트 (H1~H6)
- Body 폰트
- 크기, 줄간격, 자간

디자인 시스템 기반으로 일관된 타이포그래피를 유지합니다.

#### 3. 버튼 스타일

- Primary / Secondary 버튼
- Hover / Active 상태
- 모서리 둥글기, 패딩, 그림자

## 🤖 AI 스타일 생성의 마법

ACF CSS Manager PRO의 가장 강력한 기능은 **AI 스타일 생성**입니다.

### 사용 예시

**입력:**
```
"미니멀하고 세련된 스칸디나비안 스타일, 
화이트 베이스에 파스텔 블루 포인트"
```

**결과:**
```css
--color-primary: #6B9AC4;
--color-secondary: #E8F1F5;
--color-accent: #4A7C9B;
--color-background: #FAFCFD;
--font-heading: 'Poppins', sans-serif;
--font-body: 'Inter', sans-serif;
```

AI가 당신의 아이디어를 완벽한 CSS 변수로 변환합니다.

## 🚀 시작하기

### Step 1: 플러그인 설치
WordPress 관리자 → 플러그인 → 새로 추가 → "ACF CSS Manager" 검색

### Step 2: 색상 설정
설정 → ACF CSS Manager → Colors 탭

### Step 3: 적용 확인
프론트엔드에서 변경된 스타일 확인

**5분이면 충분합니다.**
