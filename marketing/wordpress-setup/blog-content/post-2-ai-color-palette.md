# AI로 완벽한 색상 팔레트 만들기: 실전 가이드

**카테고리:** 튜토리얼  
**태그:** AI, 색상 팔레트, 디자인, CSS  
**읽기 시간:** 7분

---

## 🎨 왜 색상 팔레트가 중요한가?

웹사이트 첫인상의 90%는 색상에서 결정됩니다. 잘못된 색상 조합은:

- 브랜드 신뢰도 하락
- 가독성 저하
- 전환율 감소

를 초래합니다.

하지만 "좋은 색상"을 고르는 건 쉽지 않죠.

---

## 🤖 AI가 색상을 선택하는 방법

ACF CSS Manager의 AI 스타일 생성은 다음 원리로 작동합니다:

### 1. 자연어 이해
사용자의 프롬프트에서 키워드 추출:
- 분위기 (모던, 클래식, 따뜻한...)
- 색상 힌트 (블루, 골드, 파스텔...)
- 참조 대상 (스칸디나비안, 일본풍...)

### 2. 색상 이론 적용
- 보색 (Complementary)
- 유사색 (Analogous)
- 삼색 조화 (Triadic)

### 3. 접근성 검증
- WCAG 2.1 대비율 확인
- 색맹 사용자 고려

## 💡 실전 프롬프트 예시

### 기업 웹사이트

**프롬프트:**
```
"전문적이고 신뢰감 있는 금융 서비스 느낌, 
네이비 블루 기반에 골드 포인트"
```

**결과:**
```
Primary: #1E3A5F (네이비)
Secondary: #D4AF37 (골드)
Accent: #4A90A4 (틸)
Background: #F8F9FA
Text: #2C3E50
```

### 카페/음식 관련

**프롬프트:**
```
"따뜻하고 아늑한 커피숍 분위기, 
브라운 계열에 크림색 배경"
```

**결과:**
```
Primary: #6F4E37 (커피 브라운)
Secondary: #A67B5B (라이트 브라운)
Accent: #C4A484 (카푸치노)
Background: #FDF5E6 (크림)
Text: #3E2723
```

### 테크/스타트업

**프롬프트:**
```
"미래지향적 AI 스타트업, 
다크 테마에 네온 퍼플과 시안 포인트"
```

**결과:**
```
Primary: #8B5CF6 (퍼플)
Secondary: #06B6D4 (시안)
Accent: #F59E0B (오렌지)
Background: #0F172A (다크)
Text: #F8FAFC
```

### 헬스케어/웰니스

**프롬프트:**
```
"깨끗하고 신선한 건강 웰니스 느낌,
민트 그린과 화이트 조합"
```

**결과:**
```
Primary: #10B981 (민트 그린)
Secondary: #6EE7B7 (라이트 민트)
Accent: #3B82F6 (블루)
Background: #FFFFFF
Text: #1F2937
```

## 📝 좋은 프롬프트 작성법

### DO ✅

1. **구체적인 분위기 묘사**
   - "럭셔리하고 고급스러운" ✓
   - "좋은 느낌" ✗

2. **참조 키워드 사용**
   - "Apple 스타일의 미니멀리즘" ✓
   - "Notion 같은 깔끔한 UI" ✓

3. **색상 힌트 제공**
   - "블루 계열 기반" ✓
   - "따뜻한 색조 위주" ✓

### DON'T ❌

1. **너무 모호한 설명**
   - "좋은 색상으로 해줘" ✗

2. **상충되는 요구**
   - "밝고 다크한 느낌" ✗

3. **너무 많은 색상 요청**
   - "빨강, 파랑, 노랑, 초록 다 넣어줘" ✗

## 🛠️ AI 생성 후 미세 조정

AI가 생성한 팔레트가 100% 완벽하지 않을 수 있습니다. 다음 단계로 미세 조정하세요:

### 1. Before/After 비교
ACF CSS Manager의 Smart Preview로 적용 전후를 비교합니다.

### 2. 특정 색상만 수정
전체가 아닌 특정 색상만 조정:
- Primary가 너무 진하면 밝기 10% 증가
- Accent가 눈에 안 띄면 채도 증가

### 3. 실제 콘텐츠로 테스트
더미 텍스트가 아닌 실제 콘텐츠로 확인하세요.

---

## 📊 색상 조합 검증 체크리스트

- [ ] Primary와 Background 대비율 4.5:1 이상
- [ ] 링크 색상이 텍스트와 구분됨
- [ ] 버튼이 충분히 눈에 띔
- [ ] 에러/성공 메시지 색상 구분 가능
- [ ] 흑백으로 봐도 계층 구분 가능

## 🎯 실습: 당신의 팔레트 만들기

**Step 1:** ACF CSS Manager PRO 설치

**Step 2:** AI 스타일 > Generate 탭

**Step 3:** 프롬프트 입력
```
"[당신의 비즈니스 설명], [원하는 분위기], [색상 힌트]"
```

**Step 4:** 생성된 결과 검토

**Step 5:** 마음에 들면 "Apply" 클릭
