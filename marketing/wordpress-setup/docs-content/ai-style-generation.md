# 🤖 AI 스타일 생성 가이드

ACF CSS Manager의 AI 기능을 사용하여 전문가 수준의 스타일을 자동으로 생성하세요.

> ⚠️ AI 스타일 생성은 PRO 버전 이상에서 사용 가능합니다.

---

## 💡 개요

AI 스타일 생성은 자연어 프롬프트를 입력하면 OpenAI GPT-4가 최적의 색상 팔레트와 타이포그래피를 자동으로 생성합니다.

### 생성 가능한 항목

| 항목 | 설명 |
|------|------|
| 🎨 색상 팔레트 | Primary, Secondary, Accent, Background, Text 색상 |
| 🔤 타이포그래피 | Heading, Body 폰트 및 크기 |
| 📐 버튼 스타일 | Border-radius, Padding, 호버 효과 |

---

## 🎯 효과적인 프롬프트 작성법

### 좋은 프롬프트 예시

```
✅ "고급스러운 블랙&골드, 법률사무소 느낌"
✅ "따뜻하고 친근한 카페 분위기, 우드톤 컬러"
✅ "모던하고 미니멀한 테크 스타트업, 블루 계열"
✅ "자연 친화적인 유기농 브랜드, 그린과 베이지"
✅ "대담하고 역동적인 스포츠 브랜드, 레드와 블랙"
```

### 피해야 할 프롬프트

```
❌ "예쁘게 해주세요" (너무 모호함)
❌ "파란색" (구체적인 분위기 없음)
❌ "#2563eb 색상으로" (AI가 생성할 필요 없음)
```

### 프롬프트 구성 요소

1. **분위기/느낌**: 고급스러운, 따뜻한, 모던한, 역동적인
2. **색상 힌트**: 블랙&골드, 블루 계열, 그린톤
3. **업종/용도**: 법률사무소, 카페, 테크 스타트업
4. **추가 키워드**: 미니멀한, 대담한, 자연 친화적

## 📖 사용 방법

### Step 1: AI 스타일 탭 접근

```
WordPress 관리자 > 설정 > ACF CSS Manager > AI 스타일
```

### Step 2: 프롬프트 입력

텍스트 입력창에 원하는 스타일을 자연어로 설명합니다.

### Step 3: 생성하기

**생성하기** 버튼을 클릭하면 AI가 스타일을 생성합니다. (약 3-5초 소요)

### Step 4: Before/After 비교

생성된 스타일을 현재 스타일과 시각적으로 비교합니다.

- 🔴 빨간색: 변경 전 값
- 🟢 녹색: 변경 후 값

### Step 5: 적용 또는 재생성

- **적용하기**: 생성된 스타일을 현재 사이트에 적용
- **다시 생성**: 다른 결과를 원할 경우 재생성
- **취소**: 변경 없이 닫기

## 🔧 설정

### API Key 설정

1. OpenAI 계정에서 API Key 발급
2. **AI 스타일 > 설정** 탭에서 API Key 입력
3. **저장** 클릭

> 💡 OpenAI API는 사용량 기반 과금입니다. 스타일 생성 1회당 약 $0.01-0.02 정도 소요됩니다.

### 생성 옵션

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| Temperature | 창의성 수준 (0-1) | 0.7 |
| Max Tokens | 응답 최대 길이 | 500 |

---

## 💾 클라우드 저장

생성된 스타일을 클라우드에 저장하여 다른 사이트에서 재사용할 수 있습니다.

1. 스타일 적용 후 **Cloud로 저장** 클릭
2. 스타일 이름 입력
3. **저장** 클릭

## ❓ FAQ

### Q: AI 생성에 비용이 드나요?

A: ACF CSS Manager 자체는 무료이지만, OpenAI API 사용료가 발생합니다. 생성 1회당 약 $0.01-0.02 정도입니다.

### Q: 생성된 스타일이 마음에 안 들면?

A: **다시 생성** 버튼으로 새로운 결과를 받거나, 프롬프트를 수정하여 재시도하세요.

### Q: 생성된 스타일을 수정할 수 있나요?

A: 네, 적용 후 일반 색상/타이포그래피 탭에서 수동으로 조정할 수 있습니다.

### Q: 오프라인에서도 사용 가능한가요?

A: 아니요, AI 생성은 OpenAI 서버와 통신이 필요합니다.
