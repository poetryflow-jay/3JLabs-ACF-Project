## 📦 설치 및 활성화

### Q: 어떤 WordPress 버전이 필요한가요?

**A:** WordPress 5.8 이상, PHP 7.4 이상이 필요합니다.

### Q: 멀티사이트에서 사용할 수 있나요?

**A:** 네, Partner 버전 이상에서 멀티사이트 네트워크를 완벽하게 지원합니다. 네트워크 전체에 스타일을 일괄 적용하거나, 각 하위 사이트별로 개별 설정이 가능합니다.

### Q: 다른 페이지 빌더와 호환되나요?

**A:** 네, 주요 페이지 빌더를 모두 지원합니다:

- ✅ Gutenberg (블록 에디터)
- ✅ Bricks Builder

- ✅ GenerateBlocks
- ✅ Kadence Blocks

## 🔑 라이센스

### Q: 라이센스 키를 어디서 찾나요?

**A:** 구매 완료 후 이메일로 라이센스 키가 발송됩니다. 또한 j-j-labs.com > 내 계정 > 주문 내역에서도 확인할 수 있습니다.

### Q: 라이센스 활성화가 안 돼요.

**A:** 다음을 확인해 주세요:
1. 라이센스 키 오타 여부 (복사-붙여넣기 권장)
2. 인터넷 연결 상태
3. 사이트 제한 초과 여부 (PRO: 무제한)
4. 방화벽에서 j-j-labs.com 차단 여부

### Q: 라이센스를 다른 사이트로 옮길 수 있나요?

**A:** 네, 현재 사이트에서 **비활성화** 후 새 사이트에서 **활성화**하면 됩니다.

1. 기존 사이트: 설정 > ACF CSS Manager > 라이센스 > **비활성화**
2. 새 사이트: 설정 > ACF CSS Manager > 라이센스 > 키 입력 > **활성화**

### Q: 라이센스가 만료되면 어떻게 되나요?

**A:** 만료 후에도 이미 적용된 스타일은 유지됩니다. 단, 다음 기능이 제한됩니다:
- ❌ 업데이트 수신 불가
- ❌ AI 스타일 생성 불가
- ❌ 클라우드 동기화 불가
- ❌ 기술 지원 불가

## 🎨 기능 관련

### Q: AI 스타일 생성에 비용이 드나요?

**A:** ACF CSS Manager PRO에는 AI 기능이 포함되어 있지만, OpenAI API 사용료가 별도로 발생합니다. 스타일 생성 1회당 약 $0.01-0.02 (약 15-30원) 정도입니다.

### Q: 내 OpenAI API Key가 필요한가요?

**A:** 네, AI 스타일 생성을 사용하려면 본인의 OpenAI 계정에서 발급받은 API Key가 필요합니다.

### Q: 클라우드에 저장된 데이터는 안전한가요?

**A:** 네, 모든 데이터는 암호화되어 전송되며, 저희 서버에서 안전하게 보관됩니다. 개인정보는 저장하지 않습니다.

### Q: CSS Tree Shaking이 뭔가요?

**A:** 사이트에서 실제로 사용하지 않는 CSS 규칙을 자동으로 제거하여 파일 크기를 줄이고 로딩 속도를 개선하는 기능입니다. PageSpeed 점수 향상에 도움이 됩니다.

## 🔧 문제 해결

### Q: 스타일이 적용되지 않아요.

**A:** 다음을 순서대로 확인해 주세요:

1. **캐시 삭제**
   - 캐싱 플러그인 캐시 삭제 (WP Rocket, LiteSpeed 등)
   - 브라우저 캐시 삭제 (Ctrl+Shift+R)
   - CDN 캐시 삭제 (Cloudflare 등)

2. **CSS 우선순위 확인**
   - 다른 플러그인이나 테마 CSS가 덮어쓰고 있을 수 있습니다
   - ACF CSS Manager > 고급 설정에서 CSS 우선순위 조정

3. **System Status 확인**
   - 설정 > ACF CSS Manager > System Status에서 오류 확인

### Q: 관리자 메뉴가 안 보여요.

**A:** 다음을 확인해 주세요:
- 관리자 권한 (Administrator) 로그인 여부
- 플러그인 활성화 여부
- 다른 플러그인과의 충돌 (순차 비활성화 테스트)

### Q: 사이트가 느려졌어요.

**A:** ACF CSS Manager는 성능에 최적화되어 있습니다. 다음을 확인해 주세요:
1. **CSS Tree Shaking 활성화** (설정 > 성능)
2. 캐싱 플러그인 사용
3. System Status에서 리소스 사용량 확인

### Q: 업데이트 후 문제가 생겼어요.

**A:** 
1. 먼저 캐시를 모두 삭제해 주세요
2. 문제가 지속되면 support@j-j-labs.com으로 연락주세요
3. 긴급 시 이전 버전으로 롤백 가능 (백업 탭)
