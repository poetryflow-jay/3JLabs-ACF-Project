# 🚀 시작하기 - ACF CSS Manager

ACF CSS Manager를 시작하는 가장 빠른 방법을 안내합니다.

## 📦 설치

### 방법 1: WordPress 관리자에서 설치

1. **플러그인 > 새로 추가** 클릭
2. "ACF CSS Manager" 검색
3. **지금 설치** 클릭
4. **활성화** 클릭

### 방법 2: ZIP 파일 업로드

1. j-j-labs.com에서 플러그인 다운로드
2. **플러그인 > 새로 추가 > 플러그인 업로드**
3. ZIP 파일 선택 후 설치
4. 활성화

---

## 🔑 라이센스 활성화 (PRO 버전)

1. **설정 > ACF CSS Manager** 이동
2. **라이센스** 탭 클릭
3. 구매 시 받은 라이센스 키 입력
4. **활성화** 버튼 클릭

> 💡 라이센스 키는 구매 확인 이메일에서 확인할 수 있습니다.

---

## 🎨 첫 번째 스타일 설정

### 색상 팔레트 설정

1. **설정 > ACF CSS Manager** 이동
2. **Colors** 탭 클릭
3. 브랜드 Primary Color 선택
4. Secondary, Accent Color 설정
5. **저장** 클릭

### 타이포그래피 설정

1. **Texts** 탭 클릭
2. Heading Font Family 선택
3. Body Font Family 선택
4. 크기 및 줄 높이 조정
5. **저장** 클릭

---

## 🤖 AI 스타일 생성 (PRO)

AI가 당신의 아이디어를 스타일로 변환합니다.

1. **AI 스타일** 탭 클릭
2. 프롬프트 입력: "고급스러운 블랙&골드, 법률사무소 느낌"
3. **생성하기** 버튼 클릭
4. Before/After 비교 확인
5. **적용하기** 클릭

---

## ☁️ 클라우드 동기화 (PRO)

설정을 클라우드에 저장하고 다른 사이트에 복제하세요.

### 내보내기

1. **Cloud** 탭 클릭
2. **클라우드로 내보내기** 클릭
3. 이름 입력 후 저장

### 가져오기

1. 다른 사이트에서 **Cloud** 탭 이동
2. **클라우드에서 가져오기** 클릭
3. 저장된 스타일 선택
4. **적용** 클릭

---

## 🔧 문제 해결

### 스타일이 적용되지 않아요

1. 캐시 플러그인 캐시 삭제
2. 브라우저 캐시 삭제 (Ctrl+Shift+R)
3. **System Status** 탭에서 오류 확인

### 라이센스 활성화 실패

1. 라이센스 키 오타 확인
2. 인터넷 연결 확인
3. 방화벽에서 j-j-labs.com 허용
