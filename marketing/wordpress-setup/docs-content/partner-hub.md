# 🏢 Partner Hub 가이드

Partner Hub는 웹 에이전시와 프리랜서를 위한 중앙 관제 시스템입니다.

> ⚠️ Partner Hub는 Partner 버전 이상에서 사용 가능합니다.

---

## 💡 개요

Partner Hub를 사용하면 수십 개의 클라이언트 사이트를 한 곳에서 관리할 수 있습니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| 🖥️ 대시보드 | 모든 클라이언트 사이트 상태 모니터링 |
| 🎨 스타일 동기화 | 마스터 스타일을 여러 사이트에 일괄 적용 |
| 🔄 원격 업데이트 | 클라이언트 사이트 원격으로 설정 변경 |
| 📊 상태 리포트 | 라이센스, 버전, 오류 현황 확인 |

---

## 🚀 시작하기

### Step 1: Neural Link 서버 설정

Partner Hub는 Neural Link 서버와 연동되어 작동합니다.

1. Neural Link가 설치된 서버 URL 확인
2. API Key 발급

### Step 2: Partner Hub 활성화

```
WordPress 관리자 > 설정 > ACF CSS Manager > Partner Hub
```

1. Neural Link URL 입력
2. API Key 입력
3. **연결** 클릭

### Step 3: 클라이언트 사이트 등록

1. **사이트 추가** 클릭
2. 클라이언트 사이트 URL 입력
3. 해당 사이트의 라이센스 키 입력
4. **등록** 클릭

---

## 🖥️ 대시보드 사용법

### 사이트 목록

| 컬럼 | 설명 |
|------|------|
| 사이트 | 클라이언트 사이트 URL |
| 상태 | 온라인/오프라인/오류 |
| 버전 | 설치된 ACF CSS 버전 |
| 마지막 동기화 | 최근 스타일 동기화 시간 |
| 액션 | 동기화, 설정, 삭제 버튼 |

### 상태 표시

- 🟢 **온라인**: 정상 작동 중
- 🟡 **업데이트 필요**: 새 버전 사용 가능
- 🔴 **오프라인**: 연결 불가
- ⚠️ **오류**: 라이센스 문제 또는 설정 오류

---

## 🎨 스타일 동기화

### 마스터 스타일 설정

1. Partner Hub에서 **마스터 스타일** 탭 클릭
2. 기본이 될 색상 팔레트 설정
3. 타이포그래피 설정
4. **마스터로 저장** 클릭

### 일괄 동기화

1. 동기화할 사이트 선택 (체크박스)
2. **선택한 사이트에 동기화** 클릭
3. 확인 다이얼로그에서 **확인** 클릭

### 개별 동기화

- 각 사이트 행의 **동기화** 버튼 클릭

## 🔗 Webhook 자동화

스타일 변경 시 외부 서비스에 자동으로 알림을 보낼 수 있습니다.

### 지원 이벤트

| 이벤트 | 설명 |
|--------|------|
| `style_updated` | 스타일 설정 변경됨 |
| `license_activated` | 새 라이센스 활성화됨 |
| `license_expired` | 라이센스 만료됨 |
| `site_error` | 사이트 오류 발생 |

### 설정 방법

1. **Webhook** 탭 클릭
2. Endpoint URL 입력
3. Secret Key 입력 (선택)
4. 알림 받을 이벤트 선택
5. **저장** 클릭

### Webhook Payload 예시

```json
{
  "event": "style_updated",
  "site_url": "https://client-site.com",
  "timestamp": "2025-12-19T10:30:00Z",
  "data": {
    "changed_settings": ["primary_color", "heading_font"],
    "user": "admin@agency.com"
  }
}
```

## 🌐 멀티사이트 네트워크

WordPress 멀티사이트 환경에서 네트워크 전체 스타일을 관리합니다.

### 네트워크 기본 스타일

1. **네트워크 관리자 > ACF CSS Manager**
2. 기본 스타일 설정
3. **네트워크에 적용** 클릭

### 사이트별 오버라이드 허용

- 각 하위 사이트가 독자적인 스타일을 가질 수 있도록 설정 가능
- **사이트 오버라이드 허용** 옵션 활성화

## ❓ FAQ

### Q: 최대 몇 개 사이트를 관리할 수 있나요?

A: Partner 라이센스 기준 무제한입니다. 단, 성능을 위해 100개 이상일 경우 Neural Link 서버 사양 업그레이드를 권장합니다.

### Q: 클라이언트 사이트에 별도 비용이 드나요?

A: Partner 라이센스에는 무제한 하위 라이센스 발행 권한이 포함됩니다.

### Q: 오프라인 사이트도 관리되나요?

A: Partner Hub는 온라인 상태의 사이트만 실시간 관리 가능합니다. 오프라인 사이트는 상태 표시로만 확인됩니다.
