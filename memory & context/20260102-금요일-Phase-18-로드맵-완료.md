## 📋 완료된 작업 요약

### 1. ACF Code Snippets Box 확장 기능

#### 1.1 조건 빌더 UI (`condition-builder.js`)
- **그룹 기반 조건 관리**: AND/OR 논리 조합 지원
- **카테고리별 조건 타입**: 기본, 페이지/포스트, 사용자, 기기/환경, 시간, WooCommerce, ACF, FacetWP, 고급
- **라이선스 기반 잠금**: Pro 전용 조건 타입 표시 및 제한
- **실시간 미리보기**: 조건 조합 텍스트 프리뷰
- **조건 테스트**: 현재 페이지에서 조건 충족 여부 확인
- **동적 옵션 로드**: AJAX를 통한 조건 값 옵션 로드

#### 1.2 넛지 마케팅 시스템 (`nudge-system.js`)
- **7가지 넛지 타입**:
  - Toast: 우하단 알림
  - Banner: 페이지 상단 배너
  - Modal: 중앙 모달 팝업
  - Tooltip: 요소 옆 툴팁
  - Spotlight: 요소 강조 + 툴팁
  - Inline: 인라인 메시지
  - Walkthrough: 단계별 가이드 투어
- **카테고리**: 온보딩, 기능안내, 업그레이드유도, 팁, 프로모션, 경고, 성공
- **트래킹**: 조회, 클릭, 닫기 행동 트래킹
- **조건 기반 표시**: 스니펫 수, 라이선스, 페이지, 플러그인 활성화 등

#### 1.3 내보내기/가져오기 시스템 (`export-import.js`)
- **내보내기**:
  - 스니펫 선택 (전체/개별)
  - JSON/ZIP 형식 지원
  - 조건 설정 포함 옵션
- **가져오기**:
  - 드래그 앤 드롭 지원
  - 진행률 표시
  - 중복 처리 옵션 (건너뛰기, 덮어쓰기, 이름변경)
  - 결과 상세 보고
- **클라우드 동기화**:
  - 동기화, 전체 업로드, 전체 다운로드 버튼

#### 1.4 서드파티 연동 (`class-acf-csb-integrations.php`)
| 플러그인 | 연동 기능 | 최소 요금제 |
|----------|----------|:-----------:|
| Advanced Custom Fields | 필드 값 조건, 반복기 행 수 | Pro Basic |
| FacetWP | Facet 선택값, 결과 수, AJAX 리프레시 | Pro Premium |
| Perfmatters | JS 지연 실행, Lazy Load 제외 | Pro Premium |
| WooCommerce | 상품 카테고리, 장바구니, 고객 이력 | Pro Basic |
| Elementor | 위젯 지원 | Pro Basic |
| Gutenberg | 블록 에디터 지원 | Free |

#### 1.5 클라우드 동기화 (`class-acf-csb-cloud-sync.php`)
- 3J Labs 클라우드 API 연동
- API 키 기반 인증
- 푸시/풀 동기화 로직
- 자동 동기화 옵션
- 크론 기반 정기 동기화 (매시간)
- 동기화 상태 표시 (synced, pending, conflict, error)

### 2. ACF CSS WooCommerce Toolkit

#### 2.1 상품 Q&A 시스템 (`class-product-qa.php`, `product-qa.js`)
- **커스텀 포스트 타입**: `product_qa`
- **상태 관리**: 답변대기, 답변완료
- **기능**:
  - 상품 탭에 Q&A 섹션 추가
  - 비공개 질문 지원
  - 게스트 질문 (이름, 이메일)
  - 관리자 답변 메타박스
  - 이메일 알림 (질문 등록, 답변 등록)
  - "도움이 됐어요" 버튼 + 트래킹
  - 스팸 방지 (1분 제한)
- **UI**:
  - 질문 폼
  - 질문/답변 목록
  - 상태 배지 (답변완료/답변대기)
  - 비공개 표시

## 🔧 요금제별 기능 분류

### Free
- Gutenberg 블록 연동
- 기본 프리셋 5개
- 10개 스니펫 제한
- 기본 조건 (위치, 포스트타입, 로그인, 기기)

### Pro Basic
- ACF (Advanced Custom Fields) 연동
- WooCommerce 기본 조건
- Elementor 위젯
- 무제한 스니펫
- 내보내기/가져오기
- 확장 조건 (URL, 분류, 역할)
- WooCommerce 프리셋

### Pro Premium
- FacetWP 연동
- Perfmatters 연동
- 클라우드 동기화
- 고급 조건 (시간, 요일, PHP)
- 조건 그룹 (AND/OR)
- 버전 히스토리
- AI 코드 제안

### Pro Unlimited / Partner / Master
- 멀티사이트 동기화
- 팀 공유
- 화이트라벨
- 개발자 도구
- 클라우드 동기화
