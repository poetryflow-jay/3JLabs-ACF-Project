# 2026-01-03 토요일 - Phase 25: Nudge Flow 메뉴 개편 및 수익화 모델 통합

## 1. 개요
사장님(Jay)의 요청에 따라 '넛지 플로우(Nudge Flow)' 모듈의 메뉴 배치와 구조를 전면 개편하고, 템플릿 생태계를 통한 수익화 모델의 기반을 구축함.

## 2. 주요 변경 사항
### Nudge Flow 메뉴 개편
- **메뉴 배치**: 기존 `JJ Admin Center` 서브메뉴에서 WooCommerce '마케팅' 메뉴 하단(Position 58)의 독립된 최상위 메뉴로 이동. 커머스 관리자의 접근성을 극대화함.
- **서브메뉴 구조화**:
  - **📊 대시보드**: 전체 통계 및 요약 (메인 진입점)
  - **🔄 워크플로우**: `jj_nudge_workflow` 포스트 타입 관리 페이지와 연결
  - **📈 분석 통계**: 개별 넛지 성과 분석 뷰
  - **🎁 템플릿 센터**: 유/무료 공유 템플릿 마켓플레이스 UI (가져오기/구매하기)
  - **⚙️ 설정**: 모듈별 환경 설정

### 수익화 모델 UI/UX 설계
- **템플릿 센터**: 유료(프리미엄) 및 무료 템플릿을 구분하여 표시.
- **판매자 정산 구조**: 직접 만든 템플릿을 마켓플레이스에 등록하고 판매 수익을 정산받을 수 있는 '판매자 등록 신청' UI 도입.
- **공유 생태계**: 사용자간 시나리오 공유를 유도하여 콘텐츠 다양성 확보.

## 3. 기술적 수정 사항
- `JJ_Master_Nudge_Flow` 클래스의 `add_admin_menu()` 메서드 수정: `add_menu_page` 및 `add_submenu_page`를 사용하여 구조 재정의.
- `render_template_center_page()` 등 신규 렌더링 메서드 추가하여 마켓플레이스 UI 기반 마련.
- `DEVELOPER_GUIDE.md` 및 `RELEASE_NOTES.md`에 관련 내용 업데이트.

## 4. 향후 과제
- 템플릿 판매 승인 및 정산 시스템 백엔드 로직 구현.
- 템플릿 센터 내 실제 공유 데이터 API 연동.
- 유료 템플릿 결제 프로세스(WooCommerce 연동) 고도화.
