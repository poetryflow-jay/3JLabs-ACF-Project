# 20260103 - Phase 27: WP Bulk Manager 그랜드 업그레이드 (v5.0.0)

## 1. 개요
WP Bulk Manager의 기능을 대폭 확장하여 단일 사이트 관리를 넘어 멀티 사이트 네트워크 및 외부 원격 사이트까지 통합 관리할 수 있는 시스템을 구축함.

## 2. 주요 변경 사항

### 2.1 멀티 사이트 지원 (Multisite Support)
- `is_multisite()`를 감지하여 전용 탭 표시.
- 네트워크 내 개별 사이트 선택을 위한 사이트 리스트 UI 추가.
- 멀티 사이트 인스톨러 및 에디터 탭 신설.

### 2.2 원격 사이트 관리 (Connected Sites)
- 시크릿 키(Secret Key)를 기반으로 한 타 워드프레스 사이트와의 보안 연결 시스템 구축.
- 원격 관리를 위한 전용 REST API 엔드포인트(`jj-bulk/v1/remote-manage`) 등록.
- `X-JJ-Bulk-Secret` 헤더를 통한 요청 검증 및 마스터 에디션 예외 처리.
- 원격 인스톨러 및 에디터 탭 신설.

### 2.3 UI/UX 개선 및 용량 안내
- 기존 'Installer', 'Editor' 탭 외에 4개의 전문 관리 탭 추가.
- 서버 및 워드프레스 설정에 따른 업로드 용량 제한 경고 문구(Red Text) 추가.
- 워드프레스 공식 문서(Increasing Maximum Upload File Size) 링크 연동.
- 파일당 최대 1GB, 전체 최대 10GB 처리 능력 명시.

## 3. 기술적 세부 사항
- **Plugin Version**: v5.0.0 (Grand Upgrade)
- **Security**: 32자리 난수 시크릿 키 생성 및 저장 (`jj_bulk_remote_secret_key`).
- **REST API**: POST 요청을 통한 원격 설치 및 관리 작업 처리.
- **Handshake**: `wp_remote_post`를 이용한 상호 연결 확인 로직 (`ajax_remote_connect`).

## 4. 향후 계획
- 원격 관리 시 HMAC-SHA256 기반의 요청 서명 기능 추가 (보안 고도화).
- 멀티 사이트 통합 에디터에서의 실시간 상태 동기화 기능 강화.
- 넛지 플로우(Nudge Flow) 업그레이드 작업 착수 예정.
