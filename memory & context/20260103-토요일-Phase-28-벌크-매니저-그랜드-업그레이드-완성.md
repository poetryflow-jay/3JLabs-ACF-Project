# 20260103 - Phase 28: WP Bulk Manager 그랜드 업그레이드 완성 (v5.0.3)

## 1. 개요
WP Bulk Manager의 기능을 대폭 확장하여 단일 사이트 관리를 넘어 멀티 사이트 네트워크 및 외부 원격 사이트까지 통합 관리할 수 있는 시스템을 완성함.

## 2. 주요 변경사항

### [v5.0.3] 그랜드 업그레이드 상세
1.  **멀티 사이트(Multisite) 통합 관리**:
    - 네트워크 내 모든 사이트 목록을 조회하고 개별 사이트를 선택하여 플러그인/테마를 통합 관리할 수 있는 UI 구현.
    - `switch_to_blog`를 활용하여 네트워크 서브사이트에 직접 플러그인을 설치하고 활성화/비활성화/삭제할 수 있는 백엔드 로직 완성.
2.  **원격 사이트(Remote Sites) 일괄 제어**:
    - 연결된 원격 사이트 목록을 DB(`jj_bulk_connected_sites`)에 저장하고 관리하는 영속성 시스템 구축.
    - REST API 엔드포인트(`/jj-bulk/v1/remote-manage`, `/jj-bulk/v1/remote-install`)를 통해 원격 사이트의 항목 조회 및 명령 전송 로직 완성.
    - **Proxy 파일 전송 시스템**: 메인 사이트에서 원격 사이트로 ZIP 파일을 직접 쏘아주는 `ajax_remote_install_proxy` 로직 구현.
3.  **보안 및 권한 강화**:
    - **일방향 관리(🛡️)**: 사장님 요청에 따라 '일방향 관리' 토글을 추가하여, 상호 동의 없이도 최초 시크릿 키 인증만으로 제어가 가능한 구조 완성.
    - **Master 무제한**: Master 에디션 감지 시 동시 작업 수량 제한을 9999개로 상향하여 대규모 환경 지원.
4.  **UI/UX 고도화**:
    - 멀티/원격 관리 탭에서 사이트 선택 후 '조회하기' 버튼으로 해당 사이트의 상태를 실시간으로 가져오는 동적 렌더링 시스템.
    - 서버 용량 제한 관련 주의 문구 및 공식 가이드 링크 강화.

## 3. 기술적 특이사항
- **Cross-site Install Flow**: Local -> Multisite -> Remote 순으로 순차적으로 설치를 진행하여 서버 부하를 최소화하고 진행 상태를 프로그레스 바로 실시간 표시.
- **REST API 인증**: `X-JJ-Bulk-Secret` 헤더를 통한 시크릿 키 검증으로 보안 통신 보장.

## 4. 향후 계획
- 넛지 플로우(Nudge Flow) 업그레이드 및 템플릿 마켓플레이스 기능 강화.
- Figma 통합 심화 및 디자인 토큰 연동.
