## 기술 노트

### 메뉴 순서 강제 지정 방법
```php
// menu_order 필터에서 배열 조작
$style_position = array_search('jj-style-guide-cockpit', $menu_order);
if ($style_position !== false) {
    unset($menu_order[$style_position]);
    $menu_order = array_values($menu_order);
}
// Dashboard 뒤에 삽입
$dashboard_position = array_search('index.php', $menu_order);
array_splice($menu_order, $dashboard_position + 1, 0, 'jj-style-guide-cockpit');
```

```javascript
);
```
