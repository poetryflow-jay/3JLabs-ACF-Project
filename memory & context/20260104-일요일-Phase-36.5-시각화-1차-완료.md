## 기술 노트

### Chart.js CDN 로드
```php
wp_enqueue_script(
    'chartjs',
    'https://cdn.jsdelivr.net/npm/chart.js@4.4.1/dist/chart.umd.min.js',
    array(),
    '4.4.1',
    true
);
```

### Gradient Card CSS
```css
.acf-nudge-stat-card:nth-child(1) {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### Chart 초기화
```javascript
new Chart(ctx, {
    type: 'line',
    data: chartData,
    options: {
        responsive: true,
        plugins: { legend: { position: 'top' } }
    }
});
```
